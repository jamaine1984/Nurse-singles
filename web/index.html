Install Nurse Singles on iPhone

iPhone installs web apps from Safari using the Share menu.

  1. Open nurse-singles.com in Safari.
  2. Tap the Share button.
  3. Choose Add to Home Screen. If you do not see it, scroll down and tap Edit Actions.
  4. Turn on Open as Web App, then tap Add.
